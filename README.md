# weather-tracker

## 🌦️ Async Weather Tracker

An asynchronous Python weather tracker that fetches real-time weather data for multiple cities concurrently using async programming.
This project demonstrates how asyncio and aiohttp can be used to improve performance when making multiple API requests.

## 🚀 Features

Fetch weather data for multiple cities simultaneously

Uses asynchronous HTTP requests for faster execution

## Displays:

Temperature

Weather condition

Humidity

Wind speed

Clean and simple CLI output

Efficient handling of multiple API calls
